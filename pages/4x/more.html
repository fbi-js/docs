# 官方模板

名称 类型 说明
factory-node 项目 Node 服务或 npm 模块
factory-web 项目 web 模板，包括 react\vue2\vue3 前端项目
factory-commands 项目 fbi 全局命令仓库

更多模板敬请期待

# 官网模板使用

# factory-commands

$ fbi add factory-commands

fbi 添加仓库后即可使用其包含的 commit、demo 等命令
